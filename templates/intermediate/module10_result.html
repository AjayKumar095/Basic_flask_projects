404 and 500 error handling.


404 ERROR

A 404 status code is returned when the requested resource is not found. Customize your 404 error page to provide a user-friendly message explaining that the requested resource could not be found or redirect them to a new location of that resource.

Generate 404

500 ERROR

A 500 status code signals an internal server error. Log detailed error messages on the server to help diagnose. On the user interface, display a friendly error message that apologizes for the inconvenience and assures users that the problem is being addressed.
